# Mermaid draw.io 式连线交互设计

## 目标与范围

参考 draw.io 固定连接点的交互，为 Mermaid Flowchart/Graph 可视化编辑器实现独立拖线控制器。节点平时隐藏连接点，悬浮时显示六个通用端口；用户可从任一端口拖线，靠近目标节点后显示其端口并自动吸附，松开后创建固定端口连线。

本次只优化新建连线。暂不支持画布边缘自动平移、点击两次连接、端口手工编辑、既有连线改接或浮动连接。标准 Mermaid DSL、后端 API、事件、数据库和保存入口保持不变。

## 六个通用端口

继续使用方向感知的三入三出物理布局和稳定 ID：`target-0..2`、`source-0..2`。名称只表示端口位于图方向的入口侧或出口侧，六点都可以发起或接收连线。Vue Flow 使用 Loose 模式查找固定端口，但 Handle 关闭原生连接能力，所有手势由自研控制器处理。

连接点直径为 14px，空闲时隐藏；鼠标悬浮节点时显示全部六点。拖线期间保留起点，并且只在当前靠近的候选目标节点上显示六点。有效吸附使用编辑器主色，无效目标使用红色反馈。

## 拖线控制器

节点根元素在指针按下时以屏幕坐标测量六个端口中心；鼠标左键落在端口中心 18px 内即发起拖线。控制器在窗口级监听 `pointermove`、`pointerup`、`pointercancel`、Escape 和失焦，移动计算通过 `requestAnimationFrame` 合并，并在结束或组件卸载时清理监听器和动画帧。

候选目标按节点外框向外扩展 24px 判定。重叠节点中，指针正下方最上层节点优先；没有直接命中时，按到节点外框的最短距离选择，平局按稳定 DOM 顺序决定。候选节点的最近端口位于 28px 吸附半径内时进入吸附状态。

拖线临时路径由覆盖画布的 SVG 绘制，使用 SmoothStep 算法并设置 `pointer-events: none`。拖线方向决定最终边的箭头方向。松开时只有有效吸附才写入草稿；空白松开、Escape、指针取消或窗口失焦均只清理临时状态。

## 连接约束

创建前拒绝已经存在的同向 `source + target` 边。相同节点的同一端口不能连接自身；相同节点使用不同端口时允许形成自环。无效端口仍显示红色吸附反馈，但松开后不修改草稿。

## 固定端口元数据

`MermaidEdge` 增加可选 `sourceHandle` 和 `targetHandle`。新建边保存用户选择的完整端口；`toVueFlowEdges()` 优先使用固定端口，旧边没有端口信息时继续按既有三入三出规则稳定分配。

序列化新增 Mermaid 可忽略的私有注释：

```mermaid
%% editor-edge-ports:
%% [
%%   {"source":"A","target":"B","sourceHandle":"source-0","targetHandle":"target-1"}
%% ]
```

元数据按唯一的 `source + target` 匹配。合法但已经陈旧的条目在读取时忽略，并在下次保存时清理；损坏、结构不合法或无法匹配到唯一边的注释不吞行，继续作为保留内容原样写回。删除节点或边后，序列化只输出仍存在且具有完整固定端口的边。

## 测试与验收

纯函数测试覆盖候选节点选择、重叠优先级、最近端口、18/24/28px 边界和确定性平局。组件与生命周期测试覆盖默认隐藏、悬浮显示、六点起线、单一目标节点、有效/无效反馈、临时路径、成功和取消路径、重复边、自环规则以及清理行为。领域与适配测试覆盖元数据往返、旧 Markdown、损坏与陈旧元数据、固定端口优先和旧边自动分配。

完成后执行前端 lint、typecheck、全量 Vitest、生产构建和 `git diff --check`，并在真实浏览器的不同缩放比例下验收起线、显示、吸附、松开、取消和重新打开后的端口保持。
